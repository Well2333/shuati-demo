# 设计

## 预期实现功能

### 答题模式

#### 常规模式

题目筛选(1)：

    常规模式：全部题目均可
    未作模式：仅出现未展示过的题目
    错题模式：仅出现做错或不会的题目
    星标模式：仅出现星标（收藏）题目
    比例模式：按照一定的比例从多个题库抽取题目

展示模式(1):

    背题模式：顺序从题库取题，直接展示答案
    刷题模式：顺序从题库取题

出题模式(0-∞)：

    乱序出题：题目顺序乱序
    乱序选项：选项顺序乱序

#### 模拟考试

题目筛选(1)：

    比例模式：按照一定的比例从多个题库抽取题目

答题限制(0-∞):

    单题时间限制
    总体时间限制
    乱序出题
    乱序选项
